「あんなに頑張ったのに、なんで分かってくれないんだろう」——人間関係で疲れる時、その多くは相手そのものより「期待とのズレ」が原因になっています。

## なぜ期待は裏切られると重くのしかかるのか

期待は、いわば「相手の行動を先取りして自分の心の中に描いた予定」です。相手が予定通りに動かなかった時、私たちは「裏切られた」と感じますが、そもそも相手は自分の心の中の予定を知らされてすらいないことがほとんどです。相手が悪いわけでも、自分が悪いわけでもなく、「期待という自分の中だけの契約」が一方的に破られたように感じているだけ、というケースが多くあります。

## 2000年前から知られている「コントロールの二分法」

この感覚を整理するのに役立つのが、古代ギリシャのストア派哲学者エピクテトスが説いた「コントロールの二分法（dichotomy of control）」という考え方です。

<SourceQuote sourceLabel="Wikipedia「Epictetus」" sourceUrl="https://en.wikipedia.org/wiki/Epictetus">
エピクテトスは、物事を「自分の力が及ぶもの」と「自分の力が及ばないもの」に分けて考えることを説いた。自分の意見、意欲、欲求、そしてどう反応するかは自分の力が及ぶ範囲にある。一方、他人の言動、評判、外部の出来事の多くは自分の力が及ばない範囲にある。この2つを混同することが、フラストレーションの根本原因になるとされる。
</SourceQuote>

相手がどう感じ、どう行動するかは、常に相手の力が及ぶ範囲にあります。自分の力が及ぶのは「自分がどう伝え、どう振る舞い、相手の反応にどう向き合うか」だけです。この考え方は、[アドラー心理学の「課題の分離」の記事](/blog/adlerian-psychology-self-axis)とも通じる部分がありますが、ここでは特に「相手への期待」というテーマに絞って掘り下げます。

## 期待を軽くする3つの実践

<StepFlow steps={[
  { title: "期待していたことを、まず自分で自覚する", description: "モヤモヤした時、「自分は何を期待していたのか」を言葉にしてみます。自覚するだけで、感情に振り回されにくくなります。" },
  { title: "期待を「伝える」ことと「叶えてもらう」ことを分ける", description: "伝えることは自分の課題、叶えるかどうかは相手の課題です。伝えた時点で、自分の役割は一旦終わりと考えます。" },
  { title: "期待が外れた時のための、自分だけの支えを持っておく", description: "相手からの反応だけに心の安定を委ねないよう、他の支えも普段から育てておきます。" },
]} />

<Callout variant="tip" title="「期待しない」は「あきらめる」ことではありません">
期待を手放すことは、相手への関心をなくすことではなく、相手の反応という「コントロールできないもの」に自分の心の安定を預けすぎないようにする工夫です。
</Callout>

## 支えを人間関係だけに集中させない

人間関係での消耗が大きくなりやすい人ほど、心の支えが「特定の相手」に集中している傾向があります。支えを複数持っておくことの効果については、[ストレス管理の基本の記事](/blog/stress-management-basics)でも詳しく紹介しています。

ココロバランスでは、人間関係以外の支え——趣味や自分の時間、他の人間関係——もあわせて記録できます。一つの関係に依存しすぎていないか、自分のポートフォリオを可視化することで気づくきっかけになります。
